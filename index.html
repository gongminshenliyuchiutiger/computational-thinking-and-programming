課程內容

本課程教授運算思維基礎概念與程式設計技巧，學生透過實作專案學習解決問題的能力。

班級社團抽籤系統

留言版系統

留言輸入、留言顯示、(進階) 儲存資料、(進階) 管理功能

五子棋遊戲系統

網頁介面 (棋盤格、標記)、遊戲邏輯 (落子判斷、勝負判斷)、玩家切換、重新開始

圖片上傳、圖片儲存、圖片顯示、(進階) 圖片管理

迷因梗圖合成系統

圖片上傳 (背景)、文字輸入、文字疊加、樣式設定、(進階) 迷因模板

Vibe Coding

關於 Vibe Coding

Vibe Coding 是由 OpenAI 共同創辦人 Andrej Karpathy 在 2025 年提出的全新開發模式，透過大型語言模型（LLM）協助，讓開發者能以自然語言 prompt 描述任務，由 AI 自動生成程式碼，減少對傳統程式技能的依賴。

這種 AI coding 的方式鼓勵開發者專注在「問題描述」與「邏輯設計」，而非語法與細節，Karpathy 形容這是一種「順著感覺走（embracing the vibes）」的寫程式體驗，讓更多非工程背景的人也能參與軟體創作。

Vibe Coding 核心理念

"程式設計不再是語法的掌握，而是意圖的表達。當我們能夠清晰地描述問題和解決方案，AI 就能將其轉化為可執行的程式碼。" — Andrej Karpathy
